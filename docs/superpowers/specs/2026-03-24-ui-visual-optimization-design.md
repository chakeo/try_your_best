# UI 视觉优化设计文档

**日期：** 2026-03-24
**方案：** 渐进式优化（方案1）
**目标：** 提升视觉设计和配色，采用简约现代风格

## 设计目标

在保持现有布局和功能的基础上，通过优化配色、间距、圆角、阴影等视觉元素，提升应用的现代感和美观度。

## 设计细节

### 1. 配色方案优化

**主题色调整：**
- 主色：从 `Colors.blue` 改为柔和的蓝色 `Color(0xFF5B8DEE)`
- 成功色：从 `Colors.green` 改为柔和的绿色 `Color(0xFF4CAF50)`
- 背景色：使用浅灰色 `Color(0xFFF5F5F5)` 替代纯白
- 卡片背景：纯白 `Colors.white`

**打卡状态颜色：**
- 已打卡圆圈：柔和绿色 `Color(0xFF4CAF50)`
- 打卡图标区域背景：浅绿色 `Color(0xFFE8F5E9)`
- 今日标识：橙色 `Color(0xFFFF9800)`

### 2. 卡片设计优化

**习惯列表卡片：**
- 增加阴影：`elevation: 2`
- 圆角：`borderRadius: 12`
- 外边距：`horizontal: 16, vertical: 8`
- 内边距：`all: 16`

**打卡图标容器：**
- 圆角：`borderRadius: 8`
- 内边距：`horizontal: 12, vertical: 8`

### 3. 日历视图优化

**日期圆圈：**
- 尺寸：36x36（从32增加到36）
- 已打卡圆圈：实心填充
- 今日边框：2px 橙色边框
- 字体大小：已打卡显示 `11`（从10增加），未打卡显示 `14`

**日历布局：**
- 增加周标题和日期之间的间距
- 优化日期网格的视觉平衡

### 4. 详情页面优化

**统计卡片：**
- 添加背景色和圆角
- 增加内边距和阴影
- 优化数字和标签的字体层级

**月份导航：**
- 增加按钮的视觉反馈
- 优化间距

### 5. 字体优化

**字体层级：**
- 习惯名称：`fontSize: 18, fontWeight: FontWeight.w500`
- 统计数字：`fontSize: 32, fontWeight: FontWeight.bold`
- 次要信息：`fontSize: 13, color: Colors.grey[600]`

## 实现范围

**需要修改的文件：**
1. `lib/main.dart` - 主题配色、习惯列表卡片
2. `lib/screens/habit_detail_screen.dart` - 日历视图、统计卡片
3. `lib/widgets/add_habit_dialog.dart` - 对话框样式（可选）

## 预期效果

- 视觉更柔和舒适
- 层次感更强
- 现代简约风格
- 保持原有功能和布局不变
